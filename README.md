## 🛠️ Tech Stack
- **FE:** Vite + React + TypeScript + Tailwind v4 + ShadcnUI
- **BE:** ElysiaJS + Prisma
- **DB:** SQLite (dev) / AWS RDS PostgreSQL (production)
- **Deploy:** AWS Lambda (BE) + AWS S3/CloudFront (FE)

## 🚀 Cara Menjalankan

### Install dependencies
```bash
bun install
```

### Jalankan Backend
```bash
cd apps/backend
bun run src/server.ts
```

### Jalankan Frontend
```bash
cd apps/frontend
bun dev
```
